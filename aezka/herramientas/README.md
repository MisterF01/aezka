# Herramientas Aezka

## Descripción
Utilidades y herramientas específicas para Aezka OS.

## Objetivo
Proporcionar herramientas de sistema accesibles con comandos en español
e interfaces intuitivas.

## Herramientas Planeadas
- Utilidades de línea de comandos en español
- Herramientas de diagnóstico del sistema
- Gestores de configuración
- Utilidades de red
- Herramientas de desarrollo
