# Componentes Personalizados de Aezka

Este directorio contiene todos los componentes específicos desarrollados para Aezka OS.

## Estructura

- **escritorio/**: Componentes del entorno de escritorio personalizado de Aezka
- **scheduler/**: Scheduler de tiempo real y gestión de procesos
- **herramientas/**: Herramientas y utilidades propias de Aezka

## Desarrollo

Todos los componentes en este directorio deben:
- Estar escritos en Rust
- Incluir comentarios claros sobre su funcionalidad
- Indicar el autor de la implementación
- Usar licencia MIT o compatible
- Seguir las convenciones de código de Redox OS
