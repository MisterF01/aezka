# Scheduler de Tiempo Real Aezka

## Descripción
Scheduler personalizado con capacidades de tiempo real para Aezka OS.

## Objetivo
Proporcionar gestión de procesos con soporte para tareas de tiempo real
manteniendo compatibilidad con el microkernel de Redox.

## Características Planeadas
- Scheduling de prioridades
- Soporte para tareas de tiempo real
- Gestión eficiente de recursos
- Métricas y monitoreo

En desarrollo - Versión 0.1.0
