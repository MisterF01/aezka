# Escritorio Aezka

## Descripción
Componentes del entorno de escritorio personalizado para Aezka OS.

## Objetivo
Crear una interfaz de usuario moderna, accesible y en español para Aezka.

## Estado Actual
En desarrollo - Versión 0.1.0

## Componentes Planeados
- Gestor de ventanas
- Panel superior/inferior
- Menú de aplicaciones
- Configuración del sistema
- Temas y personalización

## Implementador
Mister_F - 2025
Basado en tecnología Redox OS
