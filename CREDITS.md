# Créditos y Atribuciones

## Aezka OS

Aezka es un sistema operativo basado en **Redox OS**, manteniendo el respeto y
reconocimiento total al proyecto original y sus contribuidores.

## Proyecto Base: Redox OS

**Redox OS** es un sistema operativo tipo Unix escrito en Rust, con enfoque en
estabilidad, seguridad y corrección. Aezka está construido sobre esta sólida base.

### Enlaces Oficiales de Redox OS
- **Sitio Web**: https://www.redox-os.org/
- **Repositorio GitLab**: https://gitlab.redox-os.org/redox-os/redox
- **Documentación**: https://doc.redox-os.org/
- **Chat**: https://matrix.to/#/#redox-join:matrix.org

### Creador Principal de Redox OS
**Jeremy Soller** - Creador y mantenedor principal de Redox OS

### Contribuidores de Redox OS
Redox OS ha sido posible gracias a cientos de contribuidores de la comunidad
de código abierto. Ver lista completa en:
https://gitlab.redox-os.org/redox-os/redox/-/graphs/master

## Aezka - Modificaciones y Extensiones

### Equipo Aezka
- **Mister_F** - Creador y desarrollador principal de Aezka (2025)

### Objetivos de Aezka
- Sistema operativo modular con enfoque en escritorio
- Capacidades de tiempo real integradas
- Interfaz accesible en español
- Calidad profesional con código abierto

### Componentes Originales de Aezka
- Entorno de escritorio personalizado (`aezka/escritorio/`)
- Scheduler de tiempo real (`aezka/scheduler/`)
- Herramientas y utilidades propias (`aezka/herramientas/`)
