  SOLUCIÓN DE PROBLEMAS
════════════════════════════════════════════════════════════════════════════════

Error: "could not read Username"
Solución: Configura tus credenciales de GitHub:
  git config --global user.name "Tu Nombre"
  git config --global user.email "tu@email.com"

Error: "Authentication failed"
Solución: Usa un Personal Access Token en lugar de contraseña:
  1. Ve a GitHub Settings > Developer settings > Personal access tokens
  2. Genera un nuevo token con permisos de "repo"
  3. Usa el token como contraseña al hacer push

Error: "permission denied"
Solución: Verifica que tienes permisos de escritura en el repositorio

Error: "Updates were rejected"
Solución: Alguien más hizo cambios. Primero haz pull:
  git pull origin claude/create-aezka-os-vlUXd
  # Resuelve conflictos si los hay
  git push -u origin claude/create-aezka-os-vlUXd
